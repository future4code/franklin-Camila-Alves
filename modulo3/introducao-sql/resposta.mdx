## Exercício 1

a) Definição das variáveis
- **id**: identificador único (PRIMARY KEY), string de no máximo 255 caracteres;
- **name**: string de no máximo 255 caracteres, não recebe valores do=e tipo null;
- **salary**: não recebe valores de tipo null;
- **birth_date**: representação de data, não recebe valores de tipo null;
- **name**: string de no máximo 6 caracteres, não recebe valores do=e tipo null.

b) 
- **SHOW DATABASES**: apresenta as bases de dados disponíveis (nesse caso o da turma);
- **SHOW TABLES**: apresenta as tabelas disponíveis na base de dados (nesse caso o de professores, feito em aula, e o do exercício);

c) O comando **DESCRIBE** apresenta todas as propriedades das variáveis, entre elas:
- O tipo de variável com **type**;
- Se a variável aceita ou não valor **NULL**;
- Se a variável receber ou não algum tipo de chave/identificador através de uma **KEY**;

## Exercício 3

a) SELECT * FROM Actor WHERE gender = "female";

b) SELECT salary from Actor WHERE name = "Tony Ramos";

c) Zero linhas retornam, isso acontece porque gender recebe somente as opções de "male" e "female", ao solicitar "invalid" faz com que não retorne nenhum resultado.

d) 
Error Code: 1054. Unknown column 'nome' in 'field list' / **Coluna desconhecida 'nome' em 'lista de campos'**
Variável nome não existe, para corrigir o erro é necessário usar a query abaixo
SELECT id, **name** from Actor WHERE id = "002"
